# accessibility-checker

Лёгкое браузерное расширение для предварительного аудита доступности.
Расширение снимает снапшот DOM текущей страницы, прогоняет набор правил,
а затем формирует два артефакта:
- **JSON**
- **Markdown-отчёт**

## Что делает
- Сканирует активную вкладку (http/https/file; системные страницы вроде `chrome://` не поддерживаются)
- Собирает DOM-дерево
- Применяет набор проверок доступности
- Генерирует **JSON** и **Markdown** из одного и того же результата проверки  

## Быстрый старт
1. Открой `chrome://extensions/` → включи **Режим разработчика** → **Загрузить распакованное расширение** → выбери папку проекта.
2. На нужной странице нажми иконку расширения → **Сканировать**.  
   Появится анимация и затем экран результата.
3. Нажми **Скачать JSON** или **Скачать Markdown**. Кнопка **Новый скан** вернёт к стартовому экрану.

## Структура (ключевые файлы)
- `manifest.json` — конфигурация MV3  
- `popup.html / popup.css / popup.js` — UI и логика попапа  
- `ReportToMarkdown.js` — генератор Markdown-отчёта  
- `analisator/`
  - `rulesMap.js` — сопоставление тегов и правил
  - `rulesChecks.js`, `elemChecks.js` — сами проверки
  - `domTreeRunner.js` — обход DOM и сбор результата
